# Scheduling & Appointment Management System

**Tech Stack:** Next.js, Node.js, Prisma, PostgreSQL  

A platform designed to streamline scheduling for employees and events, improving coordination and minimizing conflicts through automated workflows.

---

## Features
- **Employee & Event Scheduling:** Create, edit, and manage appointments efficiently.  
- **Conflict Prevention:** Automated checks reduce overlapping schedules.  
- **Relational Data Models:** Optimized Prisma + PostgreSQL schema for fast queries.  
- **Responsive & Scalable:** Built with Next.js for performance and future expansion.
